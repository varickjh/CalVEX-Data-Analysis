# CalVEX Data Visualization Tool

An interactive R Shiny dashboard for visual analysis of data from the [California Violence Experiences Survey (CalVEX)](https://geh.ucsd.edu/california-violence-experiences-survey-calvex/). More information on the survey can be found at [https://geh.ucsd.edu/resources/](https://geh.ucsd.edu/resources/) under *California Violence Experiences Survey (CalVEX)*.

## Features

- **Violence types:** Physical Violence, Sexual Violence, Intimate Partner Violence (IPV), Sexual Violence Perpetration, Physical Violence Perpetration
- **Demographics:** Gender Identity, Age, Race/Ethnicity, Sexuality, Income Quintile, Education Level, Employment Status, Disability Status
- **Time period:** Lifetime or Past Year
- **Chart type:** Bar chart or Line chart
- **Statistics:** Percentage experiencing violence or raw count
- **Subcategories:** Toggle to view past-year subcategory breakdowns side-by-side
- **Filters:** Survey year (2020–2025) and California region (Bay, Central Valley, Mountain Valley, Northern, Southern)
- All data are weighted; an *Overall* toggle shows all respondents combined
